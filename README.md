# Adamantium
Software Web para centro medico especializado

Este proyecto comenzó a principios de 2015, en la ciudad de Neuquén, capital de la provincia de Neuquén, Argentina. Es un proyecto académico que se realizó para la materia "Practica Profesional 2" (Tesis) del Institituto de Formación y Educación Superior (IFES). 
Se basa en el modelo Domain-Driven Desing (DDD) y su implementación es en Java utilizando el framework Apache ISIS y Apache Shiro para la seguridad.

El Software permite manejar todo un centro medico.
  *Gestion de agenta
  *Gestion de turnos
  *Gestion de historia clinica
  *Alta Baja Pacientes-Medicos.
  *Reportes
  *e-mails de confirmacion de turnos
